Reflection:
I put my site on both GitHub Pages and Netlify to see how they are different. GitHub Pages was very easy. My code was already on GitHub so I just clicked settings, turned it on, and my site was working. It took maybe one minute. 

Netlify took a little longer to set up. I had to connect my GitHub account and let it see my repos. But after that it was fine. The difference I noticed is Netlify shows you what is happening when it deploys. You can see if something fails and why. GitHub Pages just does its thing without telling you much. 

I think I like Netlify more. GitHub Pages is good for simple stuff but Netlify feels like it has more options. I also learned more using it because I had to actually do things instead of just clicking one button. If I make more websites later I will probably use Netlify again. Both work for school projects but Netlify just seems better for learning how things actually work.
